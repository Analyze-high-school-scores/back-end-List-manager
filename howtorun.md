# Hướng dẫn chạy ứng dụng

## Đối với Linux/macOS

Mở terminal và di chuyển đến thư mục backend:
   ```bash
   chmod +x run.sh
   ./run.sh
   ```
## Đối với Windows

Mở command prompt và di chuyển đến thư mục backend:
```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
